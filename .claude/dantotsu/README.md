# Dantotsu — Auto-diagnostic

Diagnostics post-phase au format `DIAG-YYYY-MM-DD-description.md`.

## Template

```markdown
# DIAG-YYYY-MM-DD — Titre du diagnostic

**Phase/Epic** : Phase X / E1 — Nom
**Date** : YYYY-MM-DD

## Succès
- Ce qui a bien fonctionné
- Patterns qui ont prouvé leur efficacité

## Problèmes
- Difficultés rencontrées
- Ce qui a pris plus de temps que prévu

## Actions correctives
- Ce qu'on change pour la suite
- Améliorations de process

## Mise à jour mémoire
- Patterns validés à ajouter dans `memory/MEMORY.md`
- Pièges identifiés à documenter
```
